### What is a Graph?
- A graph is an ordered pair $G = (V, E)$, consisting of a nonempty set $V$ (the **vertices**) and a set $E$ (the **edges**) of two-element subsets of $V$.
  - For example, a graph could be $(\{a, b, c, d\}, \{\{a, b\}, \{a, c\}, \{b, c\}, \{b, d\}, \{c, d\}\})$
- Graphs that are very close to the same (but are not equal) are called **isomorphic**.
- An **isomorphism** between two graphs $G_1$ and $G_2$ is a bijection $f : V_1 \rightarrow V_2$ between the vertices of the graph such that $\{a, b\}$ is an edge in $G_1$ if and only if $\{f(a), f(b)\}$ is an edge in $G_2$
- An isomorphism, put simply, is a function with the same vertices but with different names of them. The new vertices must be connected by edges at the same points as they were in the old vertices.

### Additional Exercises

8. For which $n \geq 3$ is the graph $C_n$ bipartite?
    - A bipartite graph is one in which it is possible to divide the vertices into two sets such that no edges exist between any two vertices of the same set.
    - This means that in any given cycle of the graph, there must be an even number of vertices in that cycle.
    - This means that, for $n \geq 3$, in order for cycle $C_n$ to be bipartite, $n$ must be an even number (i.e. 4, 6, 8, etc.)
